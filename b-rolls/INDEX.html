Como gravar em 9:16 (1080×1920)

  1. OBS Studio (Windows, recomendado): Sources → Browser Source → URL = file:///C:/Users/PC/Desktop/CASAS LSF/b-rolls/A1-formulario-pedido-orcamento.html · Width=1080, Height=1920. Output → Canvas 1080×1920, 60fps, MP4 H.264 CRF 18.
  2. Chrome DevTools (rápido): abre o HTML → F12 → Ctrl+Shift+M → Responsive 1080×1920 → extensão Screencastify ou Loom para gravar.
  3. FFmpeg (CLI, Windows): ffmpeg -f gdigrab -framerate 60 -offset_x 0 -offset_y 0 -video_size 1080x1920 -i desktop -c:v libx264 -crf 18 -pix_fmt yuv420p broll.mp4
  4. Puppeteer (headless, sem mão humana): npx puppeteer-screen-recorder --url file://.../A1.html --w 1080 --h 1920 --fps 60 --duration 25 --out A1.mp4
  5. Mac: QuickTime → New Screen Recording → selecciona região 1080×1920.
  6. Cada HTML faz auto-reload ao fim da animação (loop perfeito). Grava 1-2 ciclos · corta e monta no CapCut.
  7. Instagram Reels recomenda: MP4 · 1080×1920 · 30-60fps · H.264 · <250MB. Todas as animações aqui respeitam margem de 15% topo/fundo.
